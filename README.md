# 🚗 自動車状態監視サービス - プレゼン用プロトタイプ

自動車の状態を監視・管理するサービスのプロトタイプ画面です。

## 📁 プロジェクト構成

```
プロトタイプ/
├── index.html              # ①管理画面TOP（企業ダッシュボード）
├── detail.html             # ②管理画面：車両詳細ページ
├── user-home.html          # ③エンドユーザーアプリ：ホーム画面
├── user-alert.html         # ④エンドユーザーアプリ：異常詳細画面
├── css/
│   └── style.css           # 共通スタイルシート
├── js/
│   ├── utils.js            # 共通ユーティリティ関数
│   ├── dashboard.js        # 管理画面TOP用スクリプト
│   ├── detail.js           # 車両詳細ページ用スクリプト
│   ├── user-home.js        # エンドユーザーホーム用スクリプト
│   └── user-alert.js       # 異常詳細画面用スクリプト
└── data/
    ├── vehicles.json        # 車両一覧データ
    ├── detail_CAR001.json  # 車両詳細データ
    ├── user_vehicle.json   # ユーザー車両データ
    └── alert_detail.json   # 異常詳細データ
```

## 🚀 使い方

### 1. ローカルサーバーで起動

このプロトタイプはJSONファイルを読み込むため、ローカルサーバーが必要です。

#### Pythonを使用する場合：
```bash
# Python 3の場合
python -m http.server 8000

# Python 2の場合
python -m SimpleHTTPServer 8000
```

#### Node.jsを使用する場合：
```bash
# http-serverをインストール
npm install -g http-server

# サーバーを起動
http-server -p 8000
```

#### VS Codeを使用する場合：
「Live Server」拡張機能をインストールして、HTMLファイルを右クリック→「Open with Live Server」を選択

### 2. ブラウザでアクセス

以下のURLにアクセスしてください：

- **管理画面TOP**: `http://localhost:8000/index.html`
- **車両詳細**: `http://localhost:8000/detail.html?id=CAR001`
- **ユーザーホーム**: `http://localhost:8000/user-home.html`
- **異常詳細**: `http://localhost:8000/user-alert.html`

## 📱 画面説明

### ① 管理画面TOP（企業ダッシュボード）
- KPIカードで車両群の状態を一覧表示
- 地図ビューで異常車両の位置を表示
- 整備優先度リストでリスクスコア順に表示
- 異常傾向の推移グラフ（週単位）

### ② 管理画面：車両詳細ページ
- 車両情報パネル
- 各部品の状態を色分け表示
- 故障リスク推移グラフ（過去30日）
- 整備履歴リスト
- アクションボタン（整備工場連絡、AI再診断）

### ③ エンドユーザーアプリ：ホーム画面
- 健康スコアメーター（円形ゲージ）
- 車両イラスト（各部品の状態を色で表示）
- 異常検知通知欄
- アクションボタン（詳細を見る、整備予約）

### ④ エンドユーザーアプリ：異常詳細画面
- 異常概要カード
- AI診断コメント
- 推奨アクション
- 過去比較グラフ（同部品のリスク推移）

## 🎨 使用技術

- **HTML5**: 構造
- **CSS3**: スタイリング（レスポンシブ対応）
- **JavaScript (ES6+)**: インタラクティブ機能
- **Chart.js**: グラフ描画（CDN経由）
- **JSON**: データ形式

## 📊 データ構造

### vehicles.json
車両一覧データ（管理画面TOPで使用）

### detail_{id}.json
車両詳細データ（車両詳細ページで使用）

### user_vehicle.json
ユーザー車両データ（エンドユーザーホームで使用）

### alert_detail.json
異常詳細データ（異常詳細画面で使用）

## 🔧 カスタマイズ

### データの変更
`data/`フォルダ内のJSONファイルを編集することで、表示データを変更できます。

### スタイルの変更
`css/style.css`を編集することで、デザインをカスタマイズできます。

## 📝 注意事項

- このプロトタイプはデモ用です。実際の本番環境では適切なバックエンドAPIと連携してください。
- Google Maps APIは使用していません（デモ用にSVGで簡易マップを表示）。
- データは静的なJSONファイルから読み込んでいます。
